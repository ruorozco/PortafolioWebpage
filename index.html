  • -Final Proyect: CANBUS Visualizer

    • GitHub link: CANBUS Visualizer

    Introduction:
    The Controller Area Network (CAN) is a widely used protocol in the automotive industry for internal communication between various components of a vehicle.
    In this project, we explore the possibility of controlling key vehicle functions through the manipulation of JSON packets on the CAN bus.
    Objective:
    The objective of this project is to develop a system that allows for remote control of various vehicle functions,
    such as turning lights on and off, activating windshield wipers, steering wheel movement, and starting or stopping the engine.
    All this is achieved through sending and receiving CAN messages containing commands in JSON format.
